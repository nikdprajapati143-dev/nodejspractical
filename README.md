## Installation

**1. Clone the Repository**
```bash
git clone https://github.com/your-username/nodejspractical.git
cd nodejspractical
```

**2. Backend** — open terminal in `backend/` folder
```bash
cd backend
npm install
npm run seed
npm run dev
```

**3. Frontend** — open a new terminal in `frontend/` folder
```bash
cd frontend
npm install
npm run dev
```

> Backend runs on: `http://localhost:5001`  
> Frontend runs on: `http://localhost:5173`
